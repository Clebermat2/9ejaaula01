Potenciação - O que é isso?

É um jeito chique de falar de multiplicação repetida.

Em vez de escrever: 5 x 5 x 5

A gente resume para: 5³

  • Base (o 5): É o número que se repete.
  • Expoente (o 3): Diz quantas vezes a base se repete.

Potenciação - Atalhos para Facilitar (Regra 1)

1. Para MULTIPLICAR com a mesma base:

Mantenha a base e some os expoentes. Ex: 2² · 2³ = 2²⁺³ = 2⁵

2. Para DIVIDIR com a mesma base:

Mantenha a base e diminua os expoentes. Ex: 5⁴ / 5² = 5⁴⁻² = 5²

Potenciação - Mais um Atalho (Regra 2)

3. Potência de outra potência:

Mantenha a base e multiplique os expoentes.

Exemplo: (3²)³ = 3²*³ = 3⁶

Essas regras são os principais 'macetes' para resolver contas mais rápido.

MUITO GRANDES ou MUITO PEQUENAS

É aqui que entra a Notação Científica!

Notação Científica - O Resumo dos Números

É um padrão para escrever números gigantes ou minúsculos de forma compacta.

A forma é sempre: N x 10ⁿ

  • N: Um número entre 1 e 10.
  • 10ⁿ: Uma potência de 10 que guarda a informação dos "zeros".

Notação Científica - Números GIGANTES

Exemplo: A distância da Terra ao Sol é de 150.000.000 km.

  1. A vírgula "anda" para a esquerda até sobrar só um número: 1,5
  2. Conte quantos passos a vírgula deu: 8 passos.
  3. O resumo fica: 1,5 x 10⁸ km

O expoente 8 é POSITIVO porque o número era gigante.

Notação Científica - Números MINÚSCULOS

Exemplo: O diâmetro de um fio de cabelo é 0,00007 metros.

  1. A vírgula "anda" para a direita até passar o primeiro número: 7,0
  2. Conte quantos passos a vírgula deu: 5 passos.
  3. O resumo fica: 7 x 10⁻⁵ m

O expoente -5 é NEGATIVO para mostrar que o número era minúsculo.

Fazendo Contas com Notação Científica

Multiplicação:

Multiplique os números da frente e some os expoentes. (2x10³)·(3x10⁴) = 6x10⁷

Divisão:

Divida os números da frente e diminua os expoentes. (8x10⁷)/(4x10²) = 2x10⁵

Conclusão - Resumo da Ópera

  • Potenciação: É um resumo da multiplicação repetida. Suas regras são atalhos para facilitar as contas.
  • Notação Científica: É um 'apelido' para números com muitos zeros, sejam eles gigantes ou minúsculos.
  • A grande lição: São ferramentas para deixar a matemática mais prática, organizada e menos complicada.
